# Interactive-Periodic-Table
This allows the user to hover over the different sections of the periodic table to learn about them

# How to Use
Use your mouse cursor to hover over the different colored sections of the periodic table. This will make descriptions for the various sections appear to the right. 

The switch at the top can change the periodic table to two different views. These two views have different colored sections that you can learn about.
